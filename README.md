# Binance Futures Trading Bot (Testnet)

A CLI-based Python trading bot that places orders on **Binance Futures Testnet (USDT-M)**.  
Built with a clean, reusable structure, proper input validation, logging, and error handling.

---

## Features

- Place **MARKET** and **LIMIT** orders
- Supports **BUY** and **SELL**
- Command-line interface using `argparse`
- Input validation before API calls
- Structured logging of requests and responses
- Uses **Binance Futures Testnet** (no real money)

---

## Tech Stack

- Python 3.x
- python-binance
- python-dotenv
- argparse
- logging
